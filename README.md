# 오늘의 냉장고 레시피 3개 (MVP)

자취/1인가구·신혼 타겟을 위한 "냉장고 재료 → 15분 레시피 3개 + 장보기" 웹앱.

## Stack (권장)
- Frontend: **Next.js (App Router) + Tailwind**
- Backend: **FastAPI**
- DB: Postgres(권장; Supabase/Neon 등) 또는 SQLite(초기)
- Deploy: Front(Vercel), Back(Render)

## 문서
- `PLAN.md` 전체 플랜/마일스톤
- `CHECKLIST.md` 실행 체크리스트(진행하며 체크)
- `ARCHITECTURE.md` 아키텍처/배포 구조
- `API_SPEC.md` API 스펙(JSON 스키마 포함)
- `DB_SCHEMA.md` DB 테이블 설계
- `PROMPT.md` LLM 프롬프트/검증 규칙
